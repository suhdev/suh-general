SuhCacheType

  1. type in module SuhGeneral

Creates a new cache object. The new cache will also automatically restore any previous states from the persistance storage

Usage

SuhCacheType(key, the);

Arguments

Param Type Details
key string

the application key, this key is used to store items in the persistence storage.
the Persistence

persistence storage to use.

Methods

  • store(key, value);

    Stores an element in the cache. Note: Any item with the same key will be overriden

    Parameters

    Param Type Details
    key stringobjectArray.<object>

    the item key.
    value
    (optional)
    		 stringintegerfloatboolArrayobject

    the item value.

  • disable();

    Disables the cache object such that invoking store, get, persist will result nothing. This is mainly useful upon restoring the previous object state from cache

  • enable();

    Enables the cache object

  • persist();

    Stores the current state of the cache into the loaclStorage (persistance)

  • restore();

    Restores the most recent cache state from the localStorage (persistance)

  • get(key);

    Returns an item from the cache given its key

    Parameters

    Param Type Details
    key string

    the item key to retrieve from the cache

    Returns

    anynull

    returns the requested item or null if not available

Properties

  • enabled

    boolean

    true if the cache is enabled, false otherwise.

  • data

    object

    the cache storage object
