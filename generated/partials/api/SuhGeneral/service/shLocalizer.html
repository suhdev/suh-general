shLocalizer

  1. - shLocalizerProvider
  2. service in module SuhGeneral

A module to handle localization of an Angular app by providing a way to manage translation files.

Methods

  • load(source);

    Loads either an object literal as a translation source, or a URL to get the translation data from (must return a valid JSON data)

    Parameters

    Param Type Details
    source stringobject

    the translation source (valid json)

  • ready();

    Returns a promise that will be resolved when the translation source has been loaded.

    Returns

    Promise

  • isLoaded();

    Returns either true if the source file has been loaded, false otherwise.

    Returns

    boolean

  • get(key);

    Returns a formatted string representing the translation, any string path will be parsed and the value under the specific path will be retrieved 

    var obj = {
name:{
    first:'John',
    last: 'Doe'
         }
    }
// a string path to get the first name 'name.first';

    Parameters

    Param Type Details
    key string

    a string path i.e. 'name', 'name.first', 'user.name.lastname'
