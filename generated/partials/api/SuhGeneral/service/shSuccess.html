shSuccess

  1. service in module SuhGeneral

returns a function to be used by the {SuhGeneral.DeferredRequest} service in cases of successful calls

Usage

shSuccess(d);

Arguments

Param Type Details
d deferred

the deferred object to resolve

Returns

a function that takes a single parameter (data) which is the data received from the server.
